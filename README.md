# CreditWise Loan System – Minor Project

## Overview

This project predicts whether a loan application will be approved based on applicant details using Machine Learning.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Dataset

The dataset used is **loan_approval_data.csv** which contains applicant financial and personal details.

## Features

* Data preprocessing
* Model training
* Loan approval prediction
